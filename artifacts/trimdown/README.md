# TrimDown Weight Tracker

## Category

Software Engineering and Design

## Origin

TrimDown was originally developed in CS 360: Mobile Architecture and Programming. The Android application allows users to create an account, record and review weight entries, establish a target weight, and manage notification preferences. The project uses application logic, XML layouts, and a local SQLite database to provide its core functionality.

## Capstone Enhancement

The CS 499 enhancement focused on reliability, usability, and defensive programming. The revised application validates weight entries before saving them, checks dates to prevent invalid records, and applies stronger password requirements during registration. These changes provide clearer feedback to users and reduce the chance that incomplete or unreasonable data will enter the application.
